# Outputs — Research Deliverables

Research outputs organized by timeline horizon.

## Structure

- `short_term/` — Quick deliverables (notes, summaries, preliminary analyses)
- `mid_term/` — Intermediate outputs (draft sections, experiment reports)
- `long_term/` — Final deliverables (complete papers, dissertations)
- `paper/` — Main LaTeX paper draft

## Paper

- `paper/main.tex` — Primary paper source (compile with `pdflatex main.tex`)
- `paper/main.pdf` — Compiled output

Use `writing.draft` / `writing.review` / `writing.polish` micro-skills.
Citation verification via `scripts/citation_fetch.py`.
